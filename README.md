2. Run "php -S localhost:8080" di VS Code

    -superadmin : Paling OP
    -petugas : Bisa ceklis report
    -masyarakat : Hanya bisa bikin report

4. Akun dummy:
     -mimin   | 123123 (superadmin)
     -petugas | petugas (petugas)
     -khalil  | 123 (masyarakat)
